Составные изображения и спрайты

`[Описание проблемы]
Для стандартизации доступа к суб-изображениям в спрайтах, вводятся следующие правила:

`/(для спрайтов башен/жуков)`:
`{*
	`+ спрайты, содержащие анимацию, выполняются с несколькими кадрами как по высоте, так и по ширине;
	`+ изображения поворота распологаются в следующем порядке (от первого к последнему): сначала идёт
	изображение объекта, ориентированного строго влево, каждое последующее изображение показывает
	объект, повёрнутый на минимально возможный угол по часовой стрелке относительно предыдущего;
	`+ кадры анимации в прямой последовательности располагаются слева направо;
	`+ изображения поворота располагаются сверху (от первого) вниз (к последнему);
	`+ любой отдельно взятый горизонтальный ряд мульти-изображения содержит анимацию для объекта,
	повёрнутого на один и тот же угол;
`}

`/(для спрайтов-тайловых карт)`:
`{*
	`+ тайловые карты создаются для каждого окружения (т.е. совокупности уровней, происходящих в одной
	местности) в количестве одна штука;
	`+ тайловые карты имеют размер `.256х128` пикселей (`.8х4` ячеек по `.32х32` пикселей каждая);
	`+ верхний ряд тайловой карты содержит проходимые тайлы;
	`+ второй сверху ряд содержит тайлы с участками дороги в следующем порядке:
	`{#
		`+ горизонтальная дорога;
		`+ вертикальная дорога;
		`+ "правый нижний" поворот;
		`+ "левый нижний" поворот;
		`+ "левый верхний" поворот;
		`+ "правый верхний" поворот;
		`+ дополнительное изображение горизонтальной дороги;
		`+ дополнительное изображение вертикальной дороги;
	`}
	`+ третий ряд сверху содержит `/непроходимые` тайлы;
	`+ нижний ряд используется для `/дополнительных` тайлов;
`}

Под `/проходимыми/непроходимыми` тайлами имеются ввиду соответственно тайлы, на которые можно поставить
башню и тайлы, на которых не может стоять башня. Очевидно, например, что тайлы, содержащие дорогу, являются
непроходимыми.
Место для `/дополнительных тайлов` на тайловой карте служит для хранения различных тайлов в различных картах,
но при этом для всех тайлов, стоящих на этом месте, смысл (т.е. проходимость, направление и т.д.) должен
оставаться неизменными.
