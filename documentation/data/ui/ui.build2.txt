ru.sdevteam.vinv.utils

`[Описание пакета]
Пакет комбинирует классы и интерфейсы, служащие для отображения графики
и интерфейса пользователя.

`[Основные классы]
`{*
	`+ `{ui/Screen.build2.txt}Screen` описывает произвольный экран;
	`+ `{ui/Sprite.build2.txt}Sprite` представляет точку/вектор в двумерном пространстве;
	`+ `{ui/TiledLayer.build2.txt}TiledLayer` реализует замощённый слой;
	`+ `{ui/Menu.build2.txt}Menu` предоставляет инструменты для создания и отображения меню;
	`+ `{ui/MessageBox.build2.txt}MessageBox` управляет всплывающим окном.
`}
