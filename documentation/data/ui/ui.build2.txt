ru.sdevteam.vinv.utils

`[Описание пакета]
Пакет комбинирует классы и интерфейсы, служащие для отображения графики
и интерфейса пользователя.

`[Основные классы]
`{*
	`+ `{::label}Screen` описывает произвольный экран;
	`+ `{::label}Sprite` представляет точку/вектор в двумерном пространстве;
	`+ `{::label}TiledLayer` реализует замощённый слой;
	`+ `{::label}Menu` предоставляет инструменты для создания и отображения меню;
	`+ `{::label}MessageBox` управляет всплывающим окном.
`}
